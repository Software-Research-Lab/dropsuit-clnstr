Class: clnstr

clnstr(inputopt) → {object}

new clnstr(inputopt) → {object}

The clnstr class provides a set of methods for cleaning and normalizing strings. The clnstr class accepts a raw string input and processes it by removing marks, converting the trimmed string to lowercase, and removing extra punctuation marks.
Parameters:
Name Type Attributes Default Description
input string <optional>
 null The raw string input. The input can be a sentence or word, or it can be left null to process the input passed to the constructor.
