clnstr_f(input, dispoutopt) → {object}

Remove marks, and extra marks, and convert the trimmed string to lowercase.
Parameters:
Name Type Attributes Default Description
input string The raw string input. This parameter is mandatory.
dispout boolean <optional>
 false A boolean flag that controls whether the processing output results are displayed in the terminal or not. This parameter is optional.
Source:
  • index.js, line 81
Returns:
- An object containing three methods: - txt(): Trims the input, removes punctuation marks and numbers, and converts the input to lowercase. - nmb(): Trims the input, removes punctuation marks and converts the input to lowercase. - pnc(): Trims the input, converts the input to lowercase, and removes extra punctuation marks.
Type
object
Example
const clnstr = require('./clnstr');
const input = "Hello, World! !! How are you doing today?";
const result = clnstr(input, true);
console.log(result);
// Output: { txt: 'hello world how are you doing today', nmb: 'hello world how are you doing today', pnc: 'hello, world! how are you doing today?' }
